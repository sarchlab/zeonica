# Figures

This directory contains paper-facing visual artifacts generated from the
case-study results.

- `matmul_multicore_visuals.md`: Mermaid figure sources and visual tables for
  the full-size TT-Metalium-inspired multicore matmul demo.

The figures describe the abstract data-driven mapping and per-core work
partition. They do not describe Tenstorrent hardware datapath or timing.
